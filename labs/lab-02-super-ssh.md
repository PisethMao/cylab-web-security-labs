# Lab 02 - Super SSH

## Category
General Skills

## Difficulty
Easy

## Objective
The objective of this lab is to connect to a remote server using SSH and retrieve the flag.

## Tools Used
- CYLab Webshell
- SSH
- Linux terminal

## Steps

1. Opened the `Super SSH` challenge page.
2. Clicked `Launch Instance`.
3. Copied the SSH command provided by the lab.
4. Opened the CYLab webshell.
5. Connected to the remote server using SSH.
6. Entered the provided password.
7. The server displayed the flag after successful login.
8. Copied the full flag and submitted it to CYLab.
