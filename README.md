# CYLab Web Security Labs

This repository contains my CYLab Security Academy lab write-ups for Web Security and Cybersecurity practice.

The goal of this repository is to document my learning process while solving beginner-friendly security labs, including CTF basics, Linux commands, networking, cryptography, and web exploitation.
